# Partner Skills — Installation

These three folders are populated by the official ClawHub install commands.
Run these once inside your AethirClaw container:

```bash
npx clawhub@latest install blockbeats-skill
npx clawhub@latest install rdquanyu/rootdata-crypto
npx clawhub@latest install binance/binance-skills-hub
```

OpenClaw will place the SKILL.md files here automatically.
Do not manually edit files installed by ClawHub.
